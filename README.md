PHP Dancer - Pragmatic PHP Framework
==========

Key features:
- Batch resources(CSS/JS files) loading. You add CSS/JS files in the batch and the framework load those files in one shot
- Remote functions calls
- Fast. It’s around 3 times faster than Yii
- Clean and elegant design
- Copy & run, no configuration is needed. Just copy the framework into a web accessible directory and that’s all
- Very flexible and extensible
